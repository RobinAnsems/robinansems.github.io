Project web

Een verhalenwebsite voor op reis

Opdracht:

Maak een responsive verhalenwebsite voor onderweg, de website bevat vormgegeven reisverhaaltjes. Ook bevat de website enkele microinteracties die zorgen voor betere interactie, of ervoor zorgen dat de gebruiker beter en sneller verhalen kan vinden/lezen.



De oplossing die ik heb bedacht:

Ik heb een responsive verhalenwebsite gemaakt waarbij je een verhaal kan favorieten zodat deze in je favorietenlijst komt. Ook heb ik gemaakt dat je onder een verhaal een reactie achter kunt laten. Het plaatje hieronder laat zien hoe responsive een verhaal eruit op de verhalenwebsite.




Dit is een Breakpoint graph die laat zien hoeveel ruimte er tussen zit.
